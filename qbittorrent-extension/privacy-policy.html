Privacy Policy

Last updated: April 5, 2026

Overview

qBittorrent Torrent Sender is a browser extension that helps you send .torrent files directly to your qBittorrent Web UI. We are committed to protecting your privacy.

Data Collection

This extension does not collect, transmit, or share any personal data with third parties. Specifically:

  • No analytics or tracking of any kind
  • No data is sent to any external server other than your own qBittorrent instance
  • No browsing history, search queries, or download history is recorded

Data Storage

The extension stores the following data locally on your device using the browser's built-in storage API (chrome.storage.local):

  • qBittorrent Web UI URL — the address of your qBittorrent server
  • Username and password — credentials for authenticating with your qBittorrent Web UI
  • Default save path — the directory where downloaded files are saved

This data never leaves your device except to authenticate with the qBittorrent server you have configured.

Permissions

The extension requests the following permissions:

  • downloads — to detect when .torrent files are being downloaded
  • storage — to save your settings locally
  • notifications — to display status notifications
  • host permissions — to communicate with your qBittorrent Web UI and fetch .torrent files from download URLs

Security

Your credentials are stored locally in the browser's extension storage area. We recommend:

  • Using a strong, unique password for your qBittorrent Web UI
  • Accessing your qBittorrent instance over a secure local network or VPN

Changes to This Policy

Any updates to this privacy policy will be reflected in the extension update and this page. The "Last updated" date will be revised accordingly.

Contact

If you have questions about this privacy policy, please open an issue on the project's GitHub repository.
