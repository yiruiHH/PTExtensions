# Edge Add-ons Store Listing

## Extension Name
qBittorrent Torrent Sender

## Short Description (max 132 characters)
Intercept .torrent downloads and send them directly to your qBittorrent Web UI with one click.

## Full Description
Tired of manually downloading .torrent files and uploading them to your qBittorrent Web UI? This extension streamlines the process.

**How it works:**
When you download a .torrent file, a clean confirmation popup appears showing the file details. Click "Send to qBittorrent" and the torrent is instantly added to your qBittorrent client — no extra steps needed.

**Features:**
- Automatically detects .torrent file downloads
- One-click send to your qBittorrent Web UI
- Confirmation popup — you choose whether to send or download normally
- Configurable qBittorrent server URL, credentials, and default save path
- Clean, modern dark UI
- All data stored locally — no external servers, no tracking

**Setup:**
1. Install the extension
2. Click the extension icon and enter your qBittorrent Web UI address, username, and password
3. Start downloading .torrent files — the extension handles the rest

**Requirements:**
- qBittorrent with Web UI enabled
- Web UI must be accessible from your browser (local network or VPN)

**Privacy:**
This extension does not collect or share any data. Your credentials are stored locally on your device and only used to communicate with your own qBittorrent server.
